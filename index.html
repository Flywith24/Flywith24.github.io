前言

但是 当配置更改时（如旋转屏幕），activity 会重建，但对于 UI 数据的持有者呢？

  • 开发者需要重新保存相关的信息并传递给重建的 activity ，否则开发者必须再次获取数据（通过网络请求或本地数据库）
  • 由于 UI 数据的持有者的生命周期可能比 activity 长，因此开发者还需要避免出现内存泄漏的问题
